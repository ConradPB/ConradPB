### 🏗️ Highlight Projects

#### 🥛 [MilkWize](https://github.com/conradpb) (Flagship Android App)
**The Problem:** Rural farmers lack connectivity and climate context for milk production.  
**The Solution:** An **offline-first Android app** (Kotlin/Compose) that tracks yields and correlates them with real-time **Heat Stress (THI)** data via Weather APIs.
- **Tech:** Kotlin, Coroutines, Room, Supabase, Retrofit, Android FileProvider.
- **Impact:** High-reliability data sync for low-connectivity environments.

#### 🛒 [Shopshere](https://github.com/conradpb)
AI-powered fintech e-commerce engine focused on predictive analytics and localized payment gateways.
- **Tech:** React, Fastify, PostgreSQL, Redis, Docker.

#### 📊 [Crypto Trends Visualization](https://crypto-trends-kohl.vercel.app)
Real-time financial dashboard for tracking cryptocurrency volatility and market movement.
- **Tech:** Next.js, TypeScript, WebSockets, MongoDB.
